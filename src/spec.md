# UPI Payment Gateway App

A minimal and highlight-focused UPI payment gateway application with premium UI for streamlined payment processing.

## Core Features

### Main UPI App Selection Screen
- Direct landing page with no login or profile setup required
- Prominently display supported UPI app icons in large, easily clickable buttons:
  - Google Pay
  - PhonePe
  - Paytm
  - BHIM
  - Amazon Pay
- Clean layout with bold contrast and clear visual hierarchy
- Each button redirects users to the respective UPI app with pre-filled payment details using static UPI ID: `Stake2h@upi`

### UTR Submission
- Simple, highlighted form for UTR (Unique Transaction Reference) submission after payment redirection
- Minimal distractions with focused input field
- Clear instructions and prominent submit button
- Confirmation message after successful submission
- Validation for UTR format and required fields

## Data Storage

The backend must store:
- UTR submissions with timestamps and reference numbers

## Interface Requirements
- Minimal, premium UI with bold contrast
- Clean typography and highlighted interactive elements
- Responsive layout for mobile and desktop
- Smooth transitions and animations
- Professional color scheme with clear visual hierarchy
- Streamlined user flow with no unnecessary steps
- English language content throughout
